  • Make sure that the power is OFF  before handling or assembling/disassembling the system
  • While switching on the system and also while the machine is running, do no place hands or objects near the rotating parts
  • Do not run the machine in fault conditions for longer duration. Run the machine Intermittently. Stop the machine for 5 minutes after 10minutes of testing for avoiding any  permanent damage to the machine components due to overheating.
  • For all faulty conditions the speed should be limited to 1800 rpm except for fin hitting the speed should be limited to 500 rpm. This would be  to prevent any damages to the system.
  • In case of higher rpm tests, make sure the time duration is limited.
  • Make sure to place the safety cover before running the machine at higher speeds.
  • Always ensure that the bolts in the rotor as well as the fasteners are tight and not making any unusual sounds by running at a lower RPM. After ensuring that there are no unusual noises, then the machine may be ramped to full speed.
  • Do not loosen any bolts in running conditions.
